## SQL(PostgreSQL) - ONLINE BOOKSTORE PROJECT 

** Description**
Designed and analyzed an online bookstore 
database using PostgreSQL to answer
business- related questions.

** Database Tables **
- Customers
- Books
- Orders
- Order_item
- Payments

** Key SQL concepts used **
- SELECT, WHERE ,ORDER BY
- JOINS (INNER ,LEFT)
- GROUP BY ,HAVING
- Aggregate functions
- Date functions

** Business Question Answer **
- Top selling books and authors
- monthly and yearly revenue analysis
- customer purchase behavior
-  Order frequency and average  order value

  - PostgreSQL
